# Responsive Image Gallery

A modern and responsive Image Gallery built using **HTML, CSS, and JavaScript**. The project features category-based image filtering, smooth hover animations, a fully functional lightbox with next/previous navigation, keyboard shortcuts, and a visually appealing dark-themed UI.

## Features

### Image Gallery

- Responsive image grid layout
- Dynamic image rendering using JavaScript
- Lazy image loading for better performance
- Beautiful card-based gallery design

### Hover Effects

- Smooth image zoom effect
- Overlay appearance on hover
- Card elevation animation
- Modern glassmorphism-inspired styling

### Lightbox Viewer

- Full-screen image preview
- Previous and Next navigation buttons
- Image title and caption display
- Image counter display

### Category Filtering

Users can filter images by:

- All
- Nature
- City
- People
- Architecture

### Keyboard Support

| Key  | Action              |
| ---- | ------------------- |
| ←    | Previous Image      |
| →    | Next Image          |
| Esc  | Close Lightbox      |
| Enter| Open Selected Image |

### Responsive Design

Optimized for:

- Desktop
- Laptop
- Tablet
- Mobile Devices
